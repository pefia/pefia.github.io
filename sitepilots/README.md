# SitePilots

SitePilots is a static GitHub Pages landing page for a small-business web design company.

## Stack
- Plain HTML, CSS, and vanilla JavaScript only.
- No backend, build tools, package managers, APIs, or server-side code.

## Run locally
Open `sitepilots/index.html` directly in your browser.

## Audit tool note
The URL audit scanner is a **front-end preview simulation**. It does not run real Lighthouse, backend checks, or live network tests.

## Contact email
The enquiry form uses `mailto:` and opens the user’s email app.
- Update the destination email in `sitepilots/script.js` by changing `hello@sitepilots.co.uk` in `initContactForm()`.

## Deploy to GitHub Pages
You can deploy by either:
1. Uploading/publishing the `/sitepilots` folder contents to a Pages-enabled branch/path.
2. Setting GitHub Pages to serve the folder/path that contains `/sitepilots/index.html`.

This project is designed to run as a static site without installation.
